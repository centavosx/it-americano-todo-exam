## Description

Examination for the IT-Americano

## Installation

```bash
$ npm install
```

## Database Migrations

```bash
# To generate migrations for the database schema
$ npm run migration:generate

# To run migrations
$ npm run migration:run

# To revert migrations
$ npm run migration:revert

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
